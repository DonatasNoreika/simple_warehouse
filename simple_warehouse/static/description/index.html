Simple Warehouse for your business

This module is a simple solution for the inventory organization in a several warehouses.

Simple warehouse does not interfere with a main Odoo inventory functionality

With this module you can:

Add tools (products).

Add categories.

Add warehouses.

Transfer or reserve tools.

Print lists of tools.

Add tools

Add your tools with a serial numbers, categories, notes and attach the photos and related documents.

Add Category

Add categories for a tools and print all the tools from the selected category.

Add Warehouse

Add warehouses and print all the tools from the selected warehouse.

Transfer Tools

Transfer or reserve selected quantity of the tools from one warehouse to another.

Print lists of tools

Print list of tools from selected category or warehouse.
